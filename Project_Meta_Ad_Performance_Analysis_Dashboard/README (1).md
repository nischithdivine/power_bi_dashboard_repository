# Meta Ad Performance Analysis (Power BI)

## Project Overview
This project presents a Power BI dashboard designed to analyze advertising campaign performance on Facebook and Instagram.  
It provides insights into campaign reach, engagement, conversions, and budget utilization.

The goal is to help marketing teams:
- Identify the most effective platform (Facebook vs Instagram)
- Track campaign ROI
- Optimize budget allocation
- Understand audience engagement patterns

---

## Data Scope

### In Scope
- Campaigns running on Facebook and Instagram
- Paid advertising data only

## Dashboard Insights

- **Impressions:** 216K  
- **Clicks:** 25.4K  
- **Engagements:** 29K  
- **Purchases:** 1.3K  
- **CTR:** 11.76%  
- **Engagement Rate:** 13.56%  
- **Conversion Rate:** 5.21%  
- **Purchase Rate:** 0.61%  
- **Total Budget:** 2.5M  
- **Avg Budget per Campaign:** 50.7K  

### Key Observations
- High CTR (11.76%) and engagement rate (13.56%) indicate strong ad creatives and targeting.
- Purchase rate (0.61%) is low, showing inefficiency in the conversion funnel.
- There is a drop-off from interest (clicks) to action (purchases).

## Time-Based Insights

- Peak engagement: 15–20 hours (afternoon to evening)
- Lowest engagement: 0–5 hours (early morning)
- Weekly engagement remains consistent
- Peak activity dates: 19–21 and 25–27 (June)

## Dashboard Features

- Donut Chart: Gender-based performance
- Bar Chart: Age group engagement
- Map: Country-wise performance
- Calendar Heatmap: Monthly trends
- Stacked Column: Weekly trends by ad type
- Area Chart: Hourly engagement
- Matrix: Ad type vs platform comparison


---

## Key Recommendations

1. Improve conversion strategy to address low purchase rate
2. Focus on females aged 18–30
3. Prioritize Video and Stories ad formats
4. Schedule ads during afternoon and evening
5. Allocate budget to high-performing geographies
6. Improve landing pages and retargeting strategies
